# Brain games is 5 games in your terminal

## System requirements:
    Node.js 16+ version


## How start:


    1. git clone https://github.com/lxconnerty/frontend-project-44
    
    
    2. in the repository folder run the command "make install" and npm link
    
    
    3. games start in terminal


## Game start commands:


    1. brain-games
    
    
    2. brain-calc
    
    
    3. brain-even
    
    
    4. brain-gcd
    
    
    5. brain-progression
    
    
    6. brain-prime
